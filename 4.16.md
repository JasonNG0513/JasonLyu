1.向量赋值

```{r}

sqrt(m)

exp(m)
```
2.向量运算

向量间运算
```{r}
x<-c(4,5,7)

y<-c(x,0,3,x,1,x)

m<-3*x+2*y          #循环补齐到向量最大数
```
向量特征运算 
```{r}
x<-c(4,5,7)

range(x)            #最小值最大值范围

max(x)            

min(x)

mean(x)             #向量元素均值

var(x)              #向量元素方差
```
向量排序
```{r}
x<-c(4,5,7)

sort(x)             #逆排序

order(x)            #正排序（基数序号）

order(-x)           #逆排序（基数序号）
```
矩阵排序：给定一个2*5矩阵，将第一列的元素从小到大排序，将第二列的元素从大到小排列
```{r}
x<-matrix(c(9,3,6,1,7,5,4,1,8,5),5)

x[order(x[,1],-x[,2],)
```
3.正则序列

序列生成
```{r}
seq(0,100,by=5)              

seq(from=0,by=5,length=21)    

x<-c(1,3,4)

m<-rep(x,times=3)         #向量顺序复制3次排列

m<-rep(x,each=3)          #向量元素复制3次排列
```

CH4
--------------------------------------
1.因子
```{r}
vigos <- c("care","rush","mental","disease","wrangle","spirit","commit","versus","dim","care")

vi<-factor(vigos)         #因子创建

levels(vi)                #因子水平分析
```
2.匹配分析tapply
```{r}
profit<-c(24,12,42,33,7,6,13,8,10,33)

vimean<-tapply(profit,vi,mean)            #匹配后同因子均值

vistd<-tapply(profit,vi,sd)               #匹配后同因子标准误
```

1.数组和矩阵

数组创建
```{r}
dim(z)<-c(4,5,6,7)           #4维数组，840个元素
```
