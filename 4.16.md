
--------------------------------------
1.数组和矩阵

数组创建
```{r}
dim(z)<-c(4,5,6,7)           #4维数组，840个元素

a<-c(4,2,6)

coll<-array(a,dim=c(5,6,2))  #3维数组，60个元素，变量coll分别由a循环赋值

m<-array(1:80,dim=c(2,5,2,4))             #创建元素与维数的向量

a<-array(c(1,1:3,2,2:4),dim=c(1,3,1,3))   #索引矩阵

m[a]                                      #找出索引元素

m[a]<-10                                  #元素替换
```
定义自己的函数
```{r}
name<-function(variable1,variable2) expression #定义自己的函数

a<-c(4,5)

b<-c(5,6)

z<-a%o%b          #外积

z<-outer(a,b,"*") #外积的另一种表现形式
```
矩阵转置
```{r} 
a<-array(1:3,4:6,dim=c(2,3))

m<-aperm(a,c(2,1))     #将原先的dim调换
```
矩阵相乘
```{r} 
t<-a%*%m               #上面两个矩阵相乘
```
对角矩阵
```{r} 
a<-array(1:15,dim=c(3,5))

diag(a)                #返回a矩阵的对角元素

a<-c(2,2,4)

diag(a)                #返回以a向量作为对角元素的对角矩阵
```
求解线性方程
```{r}
a<-array(3:6,dim=c(2,2))

solve(a)               #a矩阵求逆

t<-array(1,3,dim=c(2,1))

x<-solve(a,t)          #线性方程求解
```
